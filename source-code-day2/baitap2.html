Tuyển sinh Java Spring Boot phục vụ nhu cầu tuyển dụng Q4 2020

17 tháng 09,2020

Hiện nay Techmaster nhận được yêu cầu tuyển dụng lập trình viên Java Spring Boot từ tháng 12/2020 trở đi, số lượng khoảng 50 lập trình cho khoảng 10 công ty, ngân hàng khác nhau. Yêu cầu tuyển dụng khá cao, đãi ngộ tốt. Trong cuối 2019 đến đầu năm 2020, Techmaster đã đào tạo 3 lớp SpringBoot với số lượng 35 sinh viên cho 2 công ty VinID thuộc VinGroup và SmartOSC FinTech. 100% sinh viên được tuyển dụng lại và làm việc tốt. Do đó Techmaster đã có đủ kinh nghiệm và quy trình để đào tạo lập trình viên Java Spring Boot xin được việc.

Yêu cầu kinh nghiệm chung

  • Lập trình Java hướng đối tượng, collection và cấu trúc dữ liệu căn bản thành thạo
  • Lập trình Web Back End Java Spring Boot kinh nghiệm 3 tháng, hoàn thành một dự án cụ thể.
  • Quản lý phiên bản mã nguồn bằng Git
  • Sử dụng Docker căn bản để triển khai ứng dụng SpringBoot
  • Thiết kế - lập trình cơ sở dữ liệu quan hệ MySQL hoặc Postgresql
  • Đọc hiểu tiếng Anh CNTT tốt.

    • Anh Nguyễn Quang Huy, giám đốc đào tạo công nghệ VTI đến chia sẻ cơ hội việc làm Java tại Techmaster. VTI mỗi năm tuyển khoảng 200 lập trình các kỹ năng khác nhau.

    Yêu cầu bằng cấp hay kiến thức nền tảng

    Nhu cầu tuyển dụng lập trình viên Java có lớn nhưng không có nghĩa là nhà tuyển dụng chấp nhận tuyển ồ ạt, bỏ qua những bài kiểm tra, phỏng vấn đầu vào. Sẽ rất khó để một người chưa biết gì CNTT có thể học ngay lộ trình Java Spring Boot 4.5 tháng, 54 buổi vì khối lượng kiến thức quá nhiều, đặc biệt phần thực hành. Do đó các bạn sinh viên chưa thực sự vững lập trình, người chuyển ngành cần phải học trước khoá học Foundation Programming 4.5 tháng để có kỹ năng lập trình căn bản và kiến thức nền tảng CNTT.

    Lộ trình Java Full Stack 4.5 tháng, 54 buổi ~ 214 tiết học sẽ gồm 4 môn học:

  • Java căn bản 15 buổi
  • Java cấu trúc Dữ Liệu Giải Thuật 15 buổi
  • DevOps 6 buổi
  • Spring Boot 18 buổi

    • Nếu đã đăng ký học Java Full Stack 4.5 tháng, tốt nhất các bạn nên tạm dừng các hoạt động tốn thời gian, sức lực như đi làm thêm. Mỗi tuần chỉ có 3 buổi thực hành x 3 tiếng, nhưng bạn cần ôn lại kiến thức trên lớp và phải làm nhiều bài tập lập trình ở nhà và nộp để giảng viên chấm.
